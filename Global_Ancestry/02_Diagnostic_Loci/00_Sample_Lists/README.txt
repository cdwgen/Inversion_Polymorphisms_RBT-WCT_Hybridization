# These sample lists are the most high confidence individuals for diagnostic SNP discovery

####

# Species reference groups

# YCT individuals were determined by position in PCA and ADMIXTURE/ELAI analyses that determined some have small amounts of introgession
# The YCT samples in this file should be 99.9% YCT
# The WCT and RBT samples were first selected in PCA space and then unsupervised ADMIXTURE (K=2, 10 replicates) was run on the most promising samples
# The PCA cleanly split RBT vs WCT and individuals with 99.9% ancestry in one of those two groups were chosen as reference individuals

####

# Subspecies groups

# IRT and CRT individuals are harder to parse...
# First a PCA of all 99.9%+ RBT was made to see the general relationships
# Second unsupervised ADMIXTURE was run at K=2 with 10 replicates
# Replicates where the split was CRT vs IRT (based on known hatchery samples) were used to designate IRT and CRT references
